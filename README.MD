# Web-POS 

(Web Point of Sale) application  

## pre requirements
- JAVA 17
- MVN
- MYsql DBMS
## Installation 
### FrontEnd
- cd FrontEnd
- npm install
- npm run dev

### Backend
- Create DB with name __web_pos__ on mysql 
- run the command below in the project 
> mvn spring-boot:run
